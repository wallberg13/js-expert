NodeJS Stream - Ajuda processar arquivos pesados
e processar tudo sob demanda.

- Readable Streams => Ler os arquivos (fonte de dados - entrada)
- Transform Stream => Adicionar Processamento
- Writable Streams => Qualquer produto final (saida)

- Pipelines => "Funil"

=> Onde utilizar:

- ETL
- Leitura e Escrita de Arquivos Gigantes
- Geração de Relatórios
- Processamento de Audio e Video
